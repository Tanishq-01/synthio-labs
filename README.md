# AI Presentation Agent

An AI-powered voice application that generates and presents slides on any topic of your choice.

## Features

- Enter any topic and get a personalized 5-6 slide presentation
- AI voice narration for each slide
- Interactive navigation between slides
- Ask questions during the presentation
- Presentation history sidebar

## Getting Started

### Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Tech Stack

- **Frontend**: React + Vite
- **Backend**: Python (FastAPI)
- **AI**: Gemini API
